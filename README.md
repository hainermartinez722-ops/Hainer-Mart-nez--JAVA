# Sistema de Gestión de Biblioteca en Java

Este proyecto es una aplicación desarrollada en Java que permite gestionar una biblioteca básica desde la consola. El sistema permite registrar libros y usuarios, así como realizar préstamos y devoluciones.

## Descripción

El programa fue desarrollado aplicando los principios de la Programación Orientada a Objetos (POO). Está pensado como práctica académica para reforzar conceptos como clases, objetos, encapsulamiento y manejo de listas.

## Funcionalidades

- Agregar nuevos libros
- Registrar usuarios
- Prestar libros
- Devolver libros
- Listar libros disponibles
- Buscar libros por título o autor

## Tecnologías utilizadas

- Java (JDK 8 o superior)
- Programación Orientada a Objetos
- ArrayList para almacenamiento en memoria
- Aplicación de consola (CLI)

## Estructura del proyecto

src/
 ├── Main.java
 ├── Libro.java
 ├── Usuario.java
 └── Biblioteca.java

## Conceptos aplicados

- Clases y objetos
- Encapsulamiento
- Constructores
- Métodos
- Listas dinámicas (ArrayList)
- Separación de responsabilidades

## Cómo ejecutar el proyecto

1. Compilar los archivos:
   javac src/*.java

2. Ejecutar el programa:
   java src/Main
